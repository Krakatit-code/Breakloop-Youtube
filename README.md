# Breakloop-Youtube
YouTubeが同じ動画ばかりおすすめしてくるの嫌だ / Tired of YouTube recommending the same videos

## 背景
YouTubeが同じ動画ばかりおすすめに出してくるのが嫌でした。  
しかし検索ワードを無から思いついて手入力するのは大変です。  

## 解決方法
そこで、動画をざっくりカテゴリ分けし、ボタンを押すだけで検索ワードが羅列されるツールを作りました。  
さらにランダム性を持たせることで、偶然の出会いを楽しめるようにしています。
「神曲」「した結果」等頻出の特徴ワード・除外ワードもボタン一回で自動入力！
YouTubeとGoogleの動画検索に対応しています。  

## 特徴
- カテゴリごとに検索ワードを自動生成  
- ボタン操作だけで検索が可能  
- ランダム性を加えて新しい動画に出会える  
- YouTube検索とGoogle動画検索の両方に対応  

## 使い方
1. ページを開く  
2. カテゴリを選び、ボタンを押す  
3. 自動生成された検索ワードで動画検索が始まります  
4. 偶然性探索を楽しんでください
